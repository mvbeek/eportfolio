Ping:
The name is taken from sonar terminology of sending a pulse of sound and listening for the object's echo under water, is a network administration utility. Ping utility was first written by Mike Muuss in December 1983 for Ballistic Research Laboratory, now known as the US Army Research Laboratory. Ping utility is used to test reachability of a host and it measures the round-trip time for messages sent from the originating host to a destination on a network.

Pinging the host’s IP address (52.203.215.30) resulted in receiving echoes/replies back from the host as exhibit below:

Ping Results (Click here)

Security loopholes:

Receiving ping replies or echoes from a host can be used to conduct a Distributed Denial-of-Service attack, an attacker can use the ping utility to send ping requests in a very fast sequence, overwhelming the destination host with ICMP echo requests. This technique is called Ping Flood.

Ping based DDoS or Ping Flood attacks have been used in the past to bring down high profile websites such as Ebay, Yahoo, and so on, Dr. Sanjeev Kumar (2006).

Recommended mitigation:

It is recommended to block incoming traffic for Internet Control Message Protocol (ICMP) also known as Echo on the Network Firewall by denying TCP and UDP port number 7. If a complete block isn't achievable due to the business or other needs, ICMP port should only be opened/allowed for a list of limited and trusted network(s).

TraceRoute:
The Traceroute program was written by Van Jacobson in 1987 suggested by Steve Deering, Ray Belleville (2017). Traceroute is a computer diagnostic utility that is used to display possible network routes and measure packet in transit delays.

Other observations:

Other recommendations:

· The website uses protocol HTTP on TCP port 80. Communication between the web browser and the host server over HTTP is unencrypted which means data transited over HTTP network is unsecure and can be intercepted by an attacker. The data can be read, modified and leaked including passwords, secret tokens, credit card information and other sensitive information.

To mitigate this issue, it is highly recommended to use HTTPS protocol which encrypts communication end to end.
