Module 1 - Unit 6

Launching into Cyber Security May 2021

Collaborative Learning Discussion 2

Identify and discuss two security technologies and the context in which they can be employed. Your discussion could consider either network or/and software security issues.

INITIAL POST

As pertains to security technologies today, many businesses established in the first world and emerging economies are aware of the general ‘hot topic’ of cybersecurity. Some will have mitigation strategies in place, a firewall for example, however in this rapidly changing world it could be strongly advised that more is needed than simply a traditional firewall. In this discussion, we will look at Network traffic analysis and Hardware authentication as additional layers of defence.

Chirupphapa, P., Esaki, H. and Ochiai, H. (2021) suggest that due to the general increase of cyber threats, network security has become more important than ever. A network monitoring system is highly effective in protecting your local area network from cyberattacks, this continuously tracks the flow of traffic in a network. The challenge with network monitoring is the vast amount of data it generates. However, Cisco Systems Inc, (2021) highlight how Network Traffic Analysis (NTA) solves this problem. This is done by combining machine learning with behavioural analytics creating a baseline of normal network traffic and behaviour. NTA solutions build on data from other network devices such as routers, switches and firewalls to determine normal behaviour. Once the baseline has been established, NTA solution can detect and alert the IT Team when it picks up an anomaly not consistent with the normal behavioural baseline. An additional benefit of using an NTA system is that it not only detects an external threat but can also detect an internal threat and provides the visibility to mitigate the potential security event.

Multifactor authentication is another widely used security technology, known as an electronic authentication method. This method enables a user to gain access to a website or application by using the traditional username-password format. It follows by providing a one-time code presented to the user via mobile or email (B. O. ALSaleem and A. I. Alshoshan, 2021). Typically, as aforementioned, multifactor authentication requires two forms of authentication. Interestingly there is an additional, more secure form of authentication referred to as hardware authentication, this involves embedding authentication into the hardware being used. Intel’s new sixth-generation processor Core vPro combines an array of advanced algorithms simultaneously to corroborate a user’s identity. A dedicated part of the chipset is used for the security functions and authentication process. A high standard of authentication is dependent on three factors, these are; “what they know, such as a password; who they are, such as a username; and what they have, such as a token” (Mello, 2021). In this scenario, the device becomes the ‘what you have’ element.

As the Covid19 pandemic continues to ravage the world, many people are working remotely (Panetta, 2021, Figure 1), as such investing in this type of multifactor authentication can prove extremely valuable. A breach would not be possible even if hackers were to have the correct credentials, as they would need the physical device also. As cyber threats evolve it is key that we do our utmost to be able to detect and prevent any breaches using all the security tools and emerging technologies at our disposal.
