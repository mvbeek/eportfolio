Module 6 - Unit 2

Reviewing Modelling Languages and Socket Programming


This week we shall gain some hands-on experience of working with the Unified Modelling Language (UML), which is important when documenting a system design and using it to communicate with project stakeholders. You will also be introduced to the UML alternative, SysML. We will also implement a distributed communication system using Python. You will also be required to carry out some research this week, by investigating academic literature to become aware of the challenges involved in converting a monolith into a set of microservices.


On completion of this unit you will be able to:
- Discuss the steps involved in evolving a monolith system to a microservice system.
- Create a flow chart using UML or SysML notation within an open-source modelling tool.
- Implement the sockets which support distributed computing using Python, with a client enabled in one process and a   server enabled on the other.
